# 网页版使用说明

## 📄 文件说明

- **`oracle.html`** - 完整的单文件HTML应用，包含所有功能

## 🚀 使用方法

1. **直接打开**：双击 `oracle.html` 文件，在浏览器中打开即可使用

2. **资源要求**：
   - 确保 `甲骨文/` 文件夹与 `oracle.html` 在同一目录
   - 确保 `audio/` 文件夹与 `oracle.html` 在同一目录

## 📁 目录结构

```
赛博龟甲/
├── oracle.html          # 主文件
├── 甲骨文/              # 图片资源
│   ├── 人.png
│   ├── 心.png
│   └── ...
└── audio/               # 音频资源
    └── bgm.mp3.mp3
```

## ✨ 功能特点

- ✅ **单文件应用**：所有代码都在一个HTML文件中，无需服务器
- ✅ **离线可用**：不需要网络连接即可使用
- ✅ **跨平台**：支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
- ✅ **完整功能**：包含所有占卜功能、动画效果、BGM播放

## 🎯 使用流程

1. **立问**：在输入框中输入你的问题，点击"立问"
2. **进入**：长按"长按入龟甲"按钮（2秒）
3. **占卜**：点击"随问而占"开始抽取
4. **显现**：依次显示三个符号（象、势、兆）
5. **验辞**：查看解释和整体解读
6. **合甲**：20秒后自动合甲，返回立问页

## 🔧 技术说明

- **纯前端**：HTML + CSS + JavaScript
- **本地存储**：使用 `localStorage` 保存每日占卜次数
- **音频播放**：使用 HTML5 Audio API
- **动画效果**：CSS3 `@keyframes` 动画

## ⚠️ 注意事项

1. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
2. **音频播放**：某些浏览器可能阻止自动播放，需要用户交互后才能播放
3. **本地存储**：清除浏览器缓存会清除占卜记录
4. **文件路径**：不要移动 `甲骨文/` 和 `audio/` 文件夹的位置

## 🐛 常见问题

**Q: 图片不显示？**  
A: 检查 `甲骨文/` 文件夹是否与 `oracle.html` 在同一目录

**Q: 没有声音？**  
A: 检查 `audio/bgm.mp3.mp3` 文件是否存在，某些浏览器需要用户交互后才能播放音频

**Q: 占卜记录丢失？**  
A: 清除浏览器缓存会清除 `localStorage` 数据，这是正常现象

---

如有问题，请检查浏览器控制台的错误信息。
