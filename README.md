# 甲影 - 赛博甲骨占卜

这是一个交互式占卜小程序，包含**网页版**和**微信小程序版**两个版本。

## 📁 项目结构

```
赛博龟甲/
├── oracle.html          # 网页版（HTML版本）
├── 甲骨文/              # 甲骨文图片资源（两个版本共用）
│   ├── 人.png
│   ├── 心.png
│   └── ...
├── audio/               # 音频资源（两个版本共用）
│   └── bgm.mp3.mp3
├── 小程序/              # 微信小程序版本
│   ├── pages/
│   │   └── index/      # 主页面
│   ├── app.json        # 小程序配置
│   └── README.md       # 小程序使用说明
└── README.md           # 本文件
```

## 🌐 网页版（HTML版本）

### 文件位置
- `oracle.html` - 单文件HTML，包含所有HTML、CSS、JavaScript

### 使用方法
1. 直接在浏览器中打开 `oracle.html` 即可使用
2. 确保 `甲骨文/` 文件夹和 `audio/` 文件夹与 `oracle.html` 在同一目录下

### 特点
- ✅ 单文件，无需服务器
- ✅ 可直接在浏览器中运行
- ✅ 支持所有现代浏览器
- ✅ 完整的占卜功能

### 资源路径
- 图片：`甲骨文/日.png`（相对路径）
- 音频：`audio/bgm.mp3.mp3`（相对路径）

---

## 📱 微信小程序版

### 文件位置
- `小程序/` 目录下包含所有小程序文件

### 使用方法
1. 在微信开发者工具中打开 `小程序/` 目录
2. 将甲骨文图片复制到 `小程序/images/甲骨文/` 目录
3. 将音频文件复制到 `小程序/audio/bgm.mp3`
4. 编译运行

### 特点
- ✅ 微信小程序原生体验
- ✅ 可在微信中直接使用
- ✅ 支持分享和收藏
- ✅ 完整的占卜功能

### 资源路径
- 图片：`/images/甲骨文/日.png`（绝对路径）
- 音频：`/audio/bgm.mp3`（绝对路径）

### 详细说明
查看 `小程序/README.md` 获取更多信息。

## 📝 注意事项

1. **资源文件**：两个版本共用相同的图片和音频资源，但路径不同
   - 网页版使用相对路径
   - 小程序版使用绝对路径（需要复制到小程序目录）

2. **图片资源**：需要以下28个甲骨文图片
   - 象类：人、心、口、目、又、止、日、月、水、火、土、室
   - 势类：往、令、求、取、合、分、受、守、用、作
   - 兆类：吉、安、危、明、多、少

3. **音频资源**：需要背景音乐文件
   - 网页版：`audio/bgm.mp3.mp3`
   - 小程序版：`audio/bgm.mp3`（需要重命名）

## 🚀 快速开始

### 网页版
1. 双击 `oracle.html` 在浏览器中打开
2. 输入问题，开始占卜

### 小程序版
1. 打开微信开发者工具
2. 导入项目，选择 `小程序/` 目录
3. 复制资源文件到对应目录
4. 编译运行
